# 📊 JobTracker: Data-Driven Career Management

A full-stack application designed to streamline the recruitment pipeline through operational discipline and technical automation. Built to transition from a **U.S. Navy Veteran** background into the **Marketing Analytics** space.

## 🚀 The Mission
Managing a high-volume job search requires more than just spreadsheets; it requires a funnel. This tool allows for real-time tracking of application statuses, ensuring no "mission-critical" follow-up is missed.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom animations & responsive design)
* **Backend:** [Node.js](https://nodejs.org/) & [Express](https://expressjs.com/)
* **Database:** [PostgreSQL](https://www.postgresql.org/) (Relational data for application history)
* **Deployment:** [Railway](https://railway.com/)

## 📈 Key Features
* **Live Status Pulse:** Visual indicators for active application tracking.
* **Relational Mapping:** Database architecture that links companies, roles, and interview timelines.
* **Analytics Ready:** Designed to eventually calculate conversion rates from 'Applied' to 'Interview'—bringing a Marketing Analyst mindset to the job hunt.
